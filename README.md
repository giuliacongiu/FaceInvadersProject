# FaceInvadersProject
FaceInvaders is a team project in which a face recognition system was developed using the CNN architectures *ResNet101*, *DenseNet* and *EfficientNetB0*.

Data preprocessing and face cropping were performed for feature extraction, after which fine-tuning was applied to each architecture.

The performance of the models was evaluated both before and after fine-tuning, and the accuracy across architectures was analyzed and compared.
